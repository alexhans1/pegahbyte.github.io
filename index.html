Car Sharing Expense Overview

Users can use this web application to get an overview over their expenses for carsharing services. The app uses finAPI to retrieve all transactions of the past 12 month after the user has granted access to one or more bank connections. In the backend the app screens the purposes of all transactions for references to carsharing services.
The backend is implemented with Node/Express APIs. The frontend is coded in React, Bootstrap as well as Google Charts.

For PSD2 licensing reasons this project cannot be hosted publicly.

Winning Team at Bankathon 2018

In a team of four we developed an Alexa skill that could connect to the figo banking and the Franke Bornberg insurance API. The skill first identifies the customer's insurances based on their bank transactions and then matches them to contracts available over the Franke Bornberg API. With this information available users can now interact with the insurance policies asking questions like: "My bike was stolen, is it insured?".
The project that was programmed over a weekend got us the first place in the dedicated insurance track as well the grand prize each awarding €2500.
